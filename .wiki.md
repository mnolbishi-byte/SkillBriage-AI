# Project Summary
SkillBridge AI is an innovative clinical skills training platform designed to enhance the educational experience for healthcare students. By integrating with Blackboard Ultra, it provides a virtual clinical lab environment where students can practice venipuncture procedures with real-time feedback, performance tracking, and immersive simulations. The platform features a **Smart Educational Chatbot** for real-time guidance and a **Smart Emotional Support Mode** within the **AR Camera Mode** for emotional detection and adaptive AI responses. The Faculty Clinical Performance Dashboard enables educators to monitor student progress and provide targeted support.

# Project Module Description
- **Authentication**: Manages user login, logout, and session state, now with improved resilience against network issues.
- **Simulation**: Offers an interactive environment for practicing venipuncture procedures.
- **Performance Tracking**: Records simulation attempts, scores, errors, and timestamps for user review.
- **History Page**: Displays users' past simulation attempts and performance trends.
- **AI Assistant**: Provides real-time guidance and feedback during simulations.
- **Faculty Dashboard**: Allows faculty to monitor student performance with alerts and recommendations.
- **AR Camera Mode**: Features options for AR Guided Simulation, including stages for preparation, vein identification, critical error detection, needle insertion, completion, and assessment.
- **Smart Emotional Support Mode**: 
  - **Emotion Detection Engine**: Classifies emotional states based on camera feed.
  - **Emotion Indicator UI**: Displays real-time emotion indicators and toggles for emotional support.
  - **Adaptive AI Response System**: Provides supportive messages based on emotional state.
  - **Performance-Emotion Correlation**: Links simulation errors with emotional state to adjust guidance.
  - **End-of-Session Emotional Report**: Summarizes stress levels and performance metrics.
- **Smart Educational Chatbot**: 
  - Provides real-time guidance, answers questions, and clarifies clinical procedures.
  - Accessible from the main navigation and as a floating chat widget during AR simulation.
  - Uses AI text generation for responses based on course content from Blackboard Ultra.
  - **Error Handling Improvements**: Now includes smart error detection for specific issues, such as insufficient AI balance, preventing unnecessary retries and providing user-friendly messages.

# Technology Stack
- **Frontend**: React, TypeScript, Vite
- **Backend**: Python, FastAPI, SQLAlchemy
- **Database**: Atoms Cloud Database
- **Deployment**: Integrated with Blackboard Ultra for LMS capabilities

# Usage
1. **Install Dependencies**: 
   - Navigate to the backend directory and run `pip install -r requirements.txt`.
   - Navigate to the frontend directory and run `pnpm install`.
2. **Build the Application**: 
   - From the frontend directory, run `pnpm run build`.
3. **Run the Application**: 
   - Start the backend server using the appropriate command for your environment.
   - Start the frontend server using `pnpm run dev`.
